机加数采 API 文档

这份静态文档用于说明当前正式版客户端、服务端、Agent 之间的主要接口。默认部署后可通过 /api/ 目录访问。

接口概览

模块 方法 路径 作用
健康检查 GET /healthz 检查服务端是否在线。
设备管理 GET /api/device-management/overview 查询设备树、设备列表和当前实时状态。
设备管理 POST / PUT / DELETE /api/device-management/devices 新增、修改、删除设备档案。
报表 GET /api/reports/daily?from=2026-01-01&to=2026-02-01&allDevices=true 查询区间统计报表，支持单设备或全部设备。
报表 GET / PUT /api/reports/formulas 读取和维护开机率、利用率等公式。
时间线 GET /api/timeline/devices/{deviceId}?date=2026-03-23 查询某台设备某天的运行时间线。
安全 GET / POST / DELETE /api/security/* 读取用户、角色、权限并执行维护。
Agent GET /api/agent/runtime-config/{agentNodeName} Agent 拉取所属设备配置。
采集接入 POST /api/ingestion/snapshots Agent 上传机床实时快照。

典型调用

GET /api/reports/daily?from=2026-03-01&to=2026-03-23&allDevices=false&deviceId=设备GUID
GET /api/agent/runtime-config/W01-Agent
POST /api/ingestion/snapshots
